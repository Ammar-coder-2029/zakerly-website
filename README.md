# 🎓 منصة ذاكر لي (Zakerly) - صفحة هبوط تعليمية

صفحة هبوط (Landing Page) احترافية وتفاعلية تم برمجتها خصيصاً لمنصة تعليمية تستهدف تطوير مستوى الطلاب في المراحل (الابتدائية، الإعدادية، والثانوية). تم التركيز في هذا المشروع على تجربة المستخدم (UX) والتصميم المتجاوب (Responsive Design) لتقديم أفضل واجهة ممكنة للطلاب وأولياء الأمور.

---

## 🚀 المميزات (Features)

* **تصميم متجاوب 100%:** واجهة مرنة (Mobile-First) تعمل بكفاءة على شاشات الهواتف المحمولة، الأجهزة اللوحية، وشاشات الكمبيوتر.
* **تأثيرات حركية جذابة (Animations):** استخدام متقدم لخصائص التمرير (Hover) و `group-hover` لإنشاء بطاقات تفاعلية تنبض بالحياة.
* **واجهة مستخدم عصرية (UI):** ألوان مريحة للعين وتدرجات لونية (Gradients) تعطي طابعاً احترافياً وحديثاً.
* **أقسام متكاملة:**
  * شريط تنقل (Navbar) متجاوب مع تمرير سلس (Smooth Scroll).
  * قسم رئيسي (Hero Section) ترحيبي مع أزرار Call-to-Action.
  * قسم شبكي (Grid) يعرض المراحل الدراسية ومميزاتها.
  * قسم إحصائيات المنصة.
  * قسم خاص لعرض كادر المعلمين مع روابط تواصل تظهر ديناميكياً.
  * تذييل (Footer) يحتوي على نموذج تواصل متكامل ومعلومات الاتصال.

---

## 🛠️ التقنيات المستخدمة (Technologies)

* **HTML5:** لبناء الهيكل الأساسي للصفحة.
* **Tailwind CSS:** إطار عمل CSS لتنسيق الصفحة بالكامل وسرعة التطوير (تم استدعاؤه عبر CDN).
* **FontAwesome:** لإضافة أيقونات عالية الجودة وربطها بالتأثيرات الحركية.

---

## ⚙️ كيفية التشغيل (How to Run)

هذا المشروع مصمم ليكون بسيطاً ومباشراً، ولا يحتاج إلى تثبيت بيئة تشغيل معقدة:

1. قم بتحميل ملفات المشروع إلى جهازك.
2. تأكد من وجود ملف `index.html` وملف الأيقونات في مجلد `CSS/all.css` (إن وجد محلياً).
3. انقر نقراً مزدوجاً على ملف `index.html` لفتحه في أي متصفح ويب حديث (مثل Google Chrome أو Edge).
4. **ملاحظة هامة:** يجب أن تكون متصلاً بالإنترنت عند تشغيل الملف لكي يتم تحميل كلاسات Tailwind CSS بنجاح من خلال رابط الـ CDN.
