# 🎓 منصة ذاكر لي (Zakerly)

صفحة هبوط (Landing Page) تفاعلية لمنصة تعليمية تستهدف جميع المراحل الدراسية، مصممة لضمان تجربة مستخدم سلسة ومتجاوبة بالكامل مع جميع الشاشات.

* 💻 **تصفح الأكواد (Source Code):** [اضغط هنا لعرض المستودع](https://github.com/Ammar-coder-2029/zakerly-website)

---

### 🌟 أبرز المميزات
* **متجاوب 100%:** تصميم مرن (Mobile-First) يعمل بكفاءة على الهواتف وأجهزة الكمبيوتر.
* **تأثيرات حركية:** استخدام متقدم لـ Hover Animations لبطاقات تفاعلية تنبض بالحياة.
* **أقسام متكاملة:** (رئيسية، مراحل دراسية، إحصائيات، كروت المعلمين، ونموذج تواصل).

### 🛠️ التقنيات المستخدمة
* **HTML5** (الهيكل الأساسي)
* **Tailwind CSS** (التنسيق والتجاوب)
* **FontAwesome** (الأيقونات)
